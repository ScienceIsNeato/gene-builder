# Audit Reports

## What They Are

Each extraction creates a report documenting all decisions made.

## What's Included

```
GENE: lrfn1 (ENSDARG00000027602)
Link: [Ensembl gene page]

TRANSCRIPTS
✅ KEPT: lrfn1-202 [CANONICAL]
❌ FILTERED: lrfn1-201 - non-canonical

FEATURES: lrfn1-202
  5'UTR      1-224
  exon4    225-585
  exon5    586-1106
  [...]

VALIDATION CHECKLIST
[ ] Check Ensembl links above
[ ] Verify in APE

METHODOLOGY (for papers)
[Copy/paste ready text]
```

## Purpose

Makes verification straightforward. Click links to check decisions against Ensembl.

## Location

Reports saved as: `output/GENE_audit_report.txt`

## For Papers

Use the methodology section in your materials and methods.

Attach the audit report to supplementary materials if reviewers ask how you validated the sequences.
